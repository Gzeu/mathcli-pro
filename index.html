Quick Installation

Get started with MathCLI Pro in seconds

NPM (Recommended)

npm install -g mathcli-pro

NPX (Try without installing)

npx mathcli-pro calculate "2+2*5"

Requirements

  • Node.js: >= 18.0.0
  • NPM: >= 8.0.0
  • OS: Windows, macOS, Linux

Available Commands

Comprehensive toolkit for mathematical computing

calculate

Evaluate mathematical expressions

mathcli-pro calculate "sin(pi/2) + sqrt(16)"

convert

Convert between different units

mathcli-pro convert "100 cm to m"

optimize

Solve optimization problems

mathcli-pro optimize "minimize x^2+3x-5"

fetch-currency

Get real-time currency rates

mathcli-pro fetch-currency USD

plot-chart

Generate ASCII charts

mathcli-pro plot-chart 3 1 4 1 5 9

history

View and export calculation history

mathcli-pro history --export=json
